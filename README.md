এবং আদর্শ লিপি ফ্রন্ট থেকে অক্ষর একসাথে ব্যবহার করা হয়েছে ।
আর বাংলা জিকিরের জন্য SolaimanLipi এর সাথে Amiri ফন্ট একসাথে ব্যবহার করা হয়েছে।
